# QMK Simple Firmware

Descompacte e copie todo o conteúdo da pasta keyboarddelwie para dentro da pasta onde está o firmware QMK, por exemplo, c:\User\<nome_usuario>\qmk_firmware\keyboards

Compile o projeto a partir do Terminal de Comandos QMK MSYS:

qmk compile -kb keyboard_delwie -km default

Acompanhe o Artigo **[Programando Keyboard com Atmega32u4](https://garoa.net.br/wiki/Usu%C3%A1rio:Gwiethaus/QMK/Programando_Keyboard_com_Atmega32u4)** no blog do Garoa Hacker Clube para mais detalhes.
